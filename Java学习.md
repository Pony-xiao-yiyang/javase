# **Java学习**

​	简单性

​	面向对象

​	可移植性

​	高性能

​	分布式

​	动态性

​	多线程

​	安全性

​	健壮性

**2.Java三大版本**

​	JavaSE： 标准版（桌面程序，控制台开发.......）

​	JavaME：嵌入式开发（手机，小家电）

​	JavaEE：E企业级开发（web端，服务器开发...）



**3.JDK、JRE、JVM**

​	**JDK:**	Java Development Kit

​	**JRE:** 	Java Runtime Environment

​	**JVM：**JAVA Virtual Machine



**4.Java类注意事项**

​	1.每个单词的大小不能出现问题，**Java**是大小写敏感的

​	2.尽量使用英文

​	3.文件名	和	类名必须保持一致，并且首字母大写

​	4.符号是否使用了中文符号
